# CEE-paper

This repository contains the processed figure-level data and plotting scripts for all main-text and Supplementary figures in the manuscript. Plotting scripts and their corresponding data files use the same figure-number prefix.

## Scope

The repository supports reproduction of the reported figures from processed figure-level data. It does not provide an end-to-end reconstruction of the mortality estimates from raw inputs. Raw mortality records and intermediate health datasets are not included because they are subject to institutional data-use agreements.
